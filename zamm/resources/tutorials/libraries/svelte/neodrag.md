# Neodrag

## Development

Install pnpm as described [here](https://pnpm.io/installation):

```bash
$ npm install -g pnpm
```

Then install all dependencies:

```bash
$ pnpm install
$ pnpm compile
```

Now there should be compiled output:

```bash
$ ls packages/svelte/dist
index.d.ts  index.js  min
```
